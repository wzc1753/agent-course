# Demo Papers

This directory contains example papers for testing PaperGuard.

## Contents

You can add sample papers in the following formats:
- PDF files (.pdf)
- LaTeX source files (.tex)
- Plain text files (.txt)

## Example Papers

To test the system, you can use:
1. Academic papers from arXiv
2. Your own draft papers
3. Sample papers from academic conferences

## Usage

Upload any file from this directory through the PaperGuard web interface to test the review functionality.

## Note

This directory is intentionally left empty in the repository. Add your own papers for testing locally.
