================================
支持的审查模式:

1. Fast模式
   - 引用完整性检查
   - 无需API调用
   - 速度最快

2. Standard模式  
   - Fast + 外部验证
   - 元数据一致性
   - 需要网络连接

3. Full模式
   - Standard + 论断验证
   - LLM辅助验证
   - 需要OpenAI API

================================
Demo论文错误类型:

E1: 虚构引用       [23] - 不存在的论文
E2: 年份错误       [5]  - Toolformer 2024→2023
E3: DOI错误        [7]  - DOI指向错误
E4: 引用缺失       [12] - 正文有列表无
E5: 未使用引用     [18] - 列表有正文无
E6: 引用不支持     [23] - 过强论断
E7: 数字不一致     摘要15% vs 表格8%
E8: 过强论断       [5]  - "我们是第一个"

================================
依赖的外部服务:

必需:
- OpenAI API (Full模式的论断验证)
